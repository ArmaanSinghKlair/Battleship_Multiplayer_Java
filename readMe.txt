# Battleship
1. GOTO jar_files/
2. Extract both zip files
3. Goto SERVER_JAR > open powershell/cmd --> type in ( java -jar SERVER_JAR.jar)
4. Goto CLIENT_JAR and type in same command
## Note: java -jar CLIENT_JAR.jar starts a new player everytime you run that command. At least 2 players needed to play the game

### Description
This is a battleship java multiplayer with graphics and support for a mulitplayer environment using socket threading.

Server can handle multiple connections concurrently

Players can chat whoever is connected to them

Game logic correct

Players given chance to continue or exit the game
